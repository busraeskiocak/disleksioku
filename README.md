# LexiLens

Disleksili öğrenciler için AI destekli kişiselleştirilmiş okuma aracı.

https://lexilens.netlify.app

## 📹 Demo Video

## 💡 Problem
Mevcut disleksi araçları herkese aynı çözümü sunuyor. LexiLens, kullanıcının okuma paternini analiz ederek kişisel bir profil oluşturuyor.

## ✨ Özellikler
- Kalibrasyon testi (harf karışıklığı, font, renk, aralık tespiti)
- Kişisel okuma profili (UPP)
- PDF/DOCX yükleme ve okuma modu
- Profili QR kod ile paylaşma

## 🛠️ Kullanılan Teknolojiler
- React.js
- Tailwind CSS
- localStorage
- react-pdf / mammoth.js
- qrcode.react

## 🚀 Nasıl Çalıştırılır?

npm install
